# Time Series — Examples

Examples of encoding (encode) and reconstruction (encode-decode) of time-series windows using autoencoders. Each link includes a short description.

- [ts_encode.md](examples/ts_encode.md) — Encodes windows (p->k) using an autoencoder; the bottleneck provides a compact representation for downstream use.
- [ts_encode-decode.md](examples/ts_encode-decode.md) — Encodes and reconstructs (p<->k) to evaluate quality via reconstruction error.
